#### 🕵️ Deepfake Detection System (DFDA)

Multi-modal ensemble system that catches manipulated images, videos and audio. Basically a lie detector for the internet.

```python
# Actual pipeline (simplified for your sanity)
model = EnsembleOf(Xception, CNN_3D, SpectrogramAnalyzer)
result = model.predict(suspicious_media)

if result == "FAKE":
    print("🚨 Deepfake detected. The internet lied again.")
else:
    print("✅ Looks real. For now. Trust no one.")

```

*Stack: Python | PyTorch | Streamlit | Xception | 3D-CNNs | Spectrograms*
